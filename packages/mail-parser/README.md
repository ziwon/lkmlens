# @lkmlens/mail-parser (planned)

RFC822/MIME normalization: parses raw lore messages into headers
(`Message-ID`, `In-Reply-To`, `References`, `Subject`, `From`, `Date`) and a
normalized plain-text body, tolerant of malformed MIME.

Not yet implemented. See the project planning doc (kept locally, not published)
section 7.5 (Workflows) and section 8.4 (Messages).
